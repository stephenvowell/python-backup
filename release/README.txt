Python BackUp — scheduled folder backup for Windows
===================================================

Python BackUp copies a folder to a destination on a schedule you choose,
or on demand with Backup Now. Unchanged files are skipped so repeat runs stay fast.

Features
--------
- One-off or scheduled backups (interval in days + clock time)
- Incremental copies — only new or changed files are copied
- Live progress bar and log console
- CSV logs (logs\copy_success.csv, logs\copy_errors.csv)
- Remembers your last paths and schedule settings

Quick start
-----------
1. Run PythonBackUp.exe
2. Pick Source and Destination folders
3. Click Backup Now, or set a schedule and click Start Backup

Full setup steps: see SETUP.txt
